# 匹配应用

## 概述

第一步：
-一个页面两个图片，点击图片喜欢，展示该图片的喜欢数，进入下一组
-图片是随机出现，出现过的图片将不再出现，上一组喜欢过的会在下一组出现。
-无限循环，直到所有图片都已经出现过
-匹配完成后，将出现你喜欢那个图片

第二步：
-实现喜欢数记录，在匹配完后出现喜欢排行榜
-实现投递图片
-实现讨厌操作，讨厌后，将记录讨厌数
-一个用户一天只能匹配一次
-每个独立IP一天只能匹配100次

第三步
-实现登录注册
-用户自己创建匹配，生成分享url
-用户可以查看喜欢结果
-图片可以加文字
-可以投递视频
-用户可以设置权限（加密码框）
-系统自动聚合喜欢数多的匹配
-主流第三方登录
-国际化
